# OpenHansard Landing Page

A simple, clean landing page for OpenHansard - an open data repository of parliamentary records.

## Design Elements

- **Colors:**
  - Primary: #255B4B (dark teal)
  - Secondary: #E9E4DA (cream)
  - Accent: #4A738E (blue-gray)

- **Typography:**
  - Lora (serif) - for logo and headings
  - Inter (sans-serif) - for body text
  - Source Code Pro (monospace) - for code

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Google Fonts (Lora, Inter, Source Code Pro)

## Features

- Responsive design
- Clean, minimal UI
- Matches OpenHansard brand guidelines
- Fast development with Vite
- Utility-first styling with Tailwind CSS
